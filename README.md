# Taskflow – To-Do List Web App

Taskflow is a modern and responsive task management web application built using HTML, CSS, and JavaScript. It helps users organize daily activities efficiently by allowing them to add, manage, complete, and delete tasks in a clean and user-friendly interface.

The application automatically saves tasks using browser localStorage, ensuring tasks persist across sessions even after refreshing or reopening the browser.

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All / Active / Done)
- Automatic task saving using localStorage
- Responsive and modern user interface
- Smooth and user-friendly experience

## Tech Stack

- HTML
- CSS
- JavaScript
- localStorage

## Functionalities

### Add Task
Users can enter a task and add it to the task list.

### Mark Task as Completed
Tasks can be marked as completed to track progress.

### Delete Task
Users can remove unnecessary tasks from the list.

### Filter Tasks
Tasks can be filtered into:

- All Tasks
- Active Tasks
- Completed Tasks

### Data Persistence
Tasks are stored using browser localStorage and remain available after refreshing or reopening the browser.
